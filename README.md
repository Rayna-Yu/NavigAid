## To start
```bash
cd NavigAid
npm install
```

## Install dependencies
```bash
npx expo install react-native maps
npx expo install @expo/vector-icons
```

## Try it out
```bash
npx expo start
```

Make sure you have expo go installed on your phone and scan the QR.
You should now see the app on your phone
